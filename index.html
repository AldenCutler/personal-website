Some Relevant Coursework

Machine Organization and Assembly Language

GitHub Repo

Course at Worcester Polytechnic Institute covering primarily memory organization and computer system architecture, while also focusing on the Assembly language, for both Intel and AMD syntax.

There were three assignments for this course:

  • Datalab: This assignment involved using bitwise operators to write functions that accomplish certain tasks using the minimum amount of operators possible.
  • Bomblab: For this assignment, each student was given an executable "binary bomb" which we were tasked with defusing. We went about this by disassembling the executable into assembly code, and then analyzing the assembly code to determine what the bomb was doing, and how to defuse it. This assignment was a great introduction to assembly language.
  • Attacklab: The last assignment involved analyzing assembly code to find vulnerabilities in a program (for this assignment, we specifically focused on buffer overflow attacks). We then had to write an exploit string that would overflow the buffer and redirect the program to execute a different function than it was intended to.
